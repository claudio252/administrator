# Administrator app

MySQL 8 o superior

Maven

Java 8 o superior

## Installation

```
git clone https://github.com/claudio252/administrator.git
cd administrator
./mvnw clean package
```

## Usage

cd target

java -jar administrator-0.0.1-SNAPSHOT.jar

Abrir la siguiente url en el navegador: http://localhost:8080/
